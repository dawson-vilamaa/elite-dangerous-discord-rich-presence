# Elite Dangerous Rich Presence for Discord
Turns Elite Dangerous's game status into a rich presence version displaying information about your location and current ship

This was made using [electron](https://www.electronjs.org/) and was built with [electron-packager](https://github.com/electron/electron-packager). 

The program runs in the system tray. To exit, right click the icon and click "Exit".
